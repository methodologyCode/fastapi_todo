# fastapi_todo

## Функционал
```
- Получение/Добавление/Удаление тасков.
- Веб интерфейс, запросы делаются с помощью JS.
- Используется асинхронный движок для работы с базой данных + Alembic.
- Всё это реализовано через паттерны Repository + Unit of work.
```

### Установка
```
- Клонируем репозиторий
- Cоздаем в корне проекта файл .env(См. .env.example)
- Устанавливаем виртуал.окружение.
- pip3 install -r requirements.txt
- alembic upgrade head
- В корне проекта uvicorn main:app --reload
```
```
1.Repository (base_repository.py):
- AbstractRepository - абстрактный класс репозитория, который определяет методы для работы с базой данных.
- Repository - конкретная реализация репозитория, работающая с сущностью базы данных.

2.Unit of Work (unitofwork.py):
- IUnitOfWork - абстрактный класс, представляющий интерфейс для Unit of Work. Он определяет методы для создания репозиториев и управления транзакциями.
- UnitOfWork - конкретная реализация Unit of Work, которая создает репозитории для работы с базой данных, управляет транзакциями и коммитит/откатывает изменения.

3.Сервис (todo_service.py):
- ToDoService - сервис, который использует Unit of Work для взаимодействия с базой данных. Содержит методы для добавления, удаления и получения задач.

4.Зависимости (dependencies.py):
- get_todo_service - зависимость, предоставляющая экземпляр ToDoService с использованием Unit of Work. Это позволяет внедрять зависимости в другие компоненты приложения.
```
